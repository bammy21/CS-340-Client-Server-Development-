## Project Overview
This repository contains my work for CS 340, including my Project Two dashboard and supporting files. The project focused on building a data-driven dashboard for Grazioso Salvare using MongoDB, Python, and Dash.


## How do you write programs that are maintainable, readable, and adaptable?

To write maintainable and readable programs, I focus on organizing my code into smaller, reusable components. In Project One, I created a CRUD Python module that handled all database operations. This made my code more adaptable because I could reuse the same module in Project Two without rewriting database logic.

The advantage of this approach is that it separates concerns. The dashboard focuses on displaying data, while the CRUD module handles database interaction. This makes the code easier to debug, update, and scale. In the future, I could reuse this CRUD module in other applications that need to connect to MongoDB, such as web apps or analytics tools.


## How do you approach a problem as a computer scientist?

When approaching this project, I first focused on understanding the client requirements from Grazioso Salvare. I broke the problem into smaller parts, such as database setup, query design, and dashboard visualization.

This project was different from previous assignments because it required integrating multiple technologies, including MongoDB, Python, and Dash. Instead of solving isolated problems, I had to think about how all components worked together.

In the future, I would continue using strategies like breaking problems into smaller steps, testing each part individually, and designing flexible database structures that can support different types of queries.

## What do computer scientists do, and why does it matter?

Computer scientists design and build systems that help organizations manage and analyze data more efficiently. In this project, I created a dashboard that allows users to filter and visualize animal data quickly.

This type of work is important because it helps companies like Grazioso Salvare make better decisions. For example, they can quickly identify animals that meet specific rescue requirements. This improves efficiency, saves time, and supports better outcomes in their operations.
